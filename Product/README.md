### Prerequisites
- Node.js (versi 14 atau lebih baru)
- npm atau yarn

### Langkah Instalasi

1. Clone repository atau download project

2. Masuk ke folder Product
```bash
cd Product
```

3. Install dependencies
```bash
npm install
```

4. Jalankan development server
```bash
npm run dev
```

5. Buka browser dan akses `http://localhost:5173`
